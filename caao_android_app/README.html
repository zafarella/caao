Build instructions

The project doesn't have any specific settings. You can import it into the environment and correct the path to the android sdk jar file. The rest should work fine.

Extending the project

There is an some task which wasn't completed yet, you can find all of them in source code. to see the list of tasks, open the todo window in eclipse. Please note it's up to you how you will extend the project, but the basic and most important functionality is implemented. The code will give you better view of the application implementation.

Directory structure

The structure of directory belongs to the style of the android project directory structure.

Diagrams and code profiling reports

The reports made by CodePro could be found in code_pro_reports folder.
